# Backlog Idees Produit

Ce dossier centralise les idees produit pour ne rien perdre entre les sprints.

## Index
- `docs/backlog/FEATURE_BACKLOG.md`: vue globale priorisee.
- `docs/backlog/GOALS_IDEAS_BACKLOG.md`: objectifs d'epargne (ex: "Thailande 2027").
- `docs/backlog/CURRENCY_IDEAS_BACKLOG.md`: multi-devise V2.
- `docs/backlog/TRANSACTIONS_IDEAS_BACKLOG.md`: UX transaction + remboursements.

## Regle d'usage
- Ajouter une idee dans le fichier du theme.
- Donner un statut:
  - `Now`: a faire maintenant
  - `Next`: prochain lot
  - `Later`: a garder pour plus tard
- Ajouter une mini definition of done pour les idees importantes.
